# DRPodcast
[<img alt="Recycled DR Lyd logo" src="https://xmpl.dk/DRPodcast/assets/icon-logo-drlyd-recycled.svg" width="150">](https://xmpl.dk/DRPodcastShell)

A rewrite of my first [DRPodcast](https://github.com/briped/briped.github.io) PowerShell module. There's no real goal except for me to try out things, make the code cleaner and, hopefully, also make a Python version, so I can learn some Python along the way.

The purpose of this is to generate podcast feeds for the DR Lyd shows that are either delayed or completely locked away in the DR Lyd app.

The reason is that DR decided to delay publishing some shows, and completely remove others, from third party Podcast players, forcing you to use their DR Lyd app, rather than your preferred Podcast app.
